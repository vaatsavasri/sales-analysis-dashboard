# Sales Performance Analysis Dashboard

Project Overview
This project analyzes sales data to identify revenue trends, top-performing products,
regional performance, and customer behavior using Python.

Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit

 Key Features
- Total revenue and profit analysis
- Monthly sales trend visualization
- Region-wise sales performance
- Interactive Streamlit dashboard

How to Run
open teriminal in vs code 
  pip install -r requirements.txt
  python -m streamlit run app.py
